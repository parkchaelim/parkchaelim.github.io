# 👗 Outfit Archive

세로 긴 캡처 이미지를 효율적으로 보관하고 태그로 검색하는 모바일 웹앱입니다.

## 주요 기능

### 📸 이미지 보관
- **빠른 저장**: 이미지 추가 → 태그 없이 즉시 저장
- **세로 이미지 지원**: contain 모드로 전체 코디가 보이는 썸네일
- **형식**: PNG, JPG, WebP 등 모든 이미지 포맷 지원

### 🔍 검색
- **텍스트 검색**: 메모, 태그 등에서 빠르게 검색
- **배지 기반 태그 선택**: 클릭으로 태그 추가/제거
- **AND/OR 필터**: 모두 포함 또는 하나라도 포함 선택 가능
- **최신순/오래된순 정렬**

### 🏷️ 태그 관리
- **카테고리 기반**: 스타일, 색상, 아이템으로 분류된 배지
- **최근/자주 쓰는 태그**: 빈도순 추천
- **새 태그 즉시 생성**: 검색 중 없는 태그 추가 가능

### 🖼️ 이미지 뷰어
- **풀스크린 보기**: 카드 탭 시 크게 확인
- **좌우 스와이프**: 이미지 네비게이션
- **편집/삭제**: 뷰어 내에서 직접 수정

### 💾 백업
- **JSON 내보내기**: 전체 데이터 다운로드
- **가져오기**: 병합 또는 덮어쓰기 선택 가능
- **로컬 저장**: IndexedDB (자동 저장)

---

## 사용법

### 1️⃣ 이미지 추가
- **보관함** 탭 → **+ 이미지 추가**
- 파일 선택, 드래그&드롭, 또는 Ctrl+V로 붙여넣기
- "태그 없이 저장" 체크 시 즉시 저장

### 2️⃣ 태그 추가 (나중에)
- 보관함에서 이미지 탭 → 편집 화면
- **+ 태그 추가** 버튼 → 배지 선택 또는 새로 생성
- 메모 추가 후 **저장**

### 3️⃣ 검색
- **검색** 탭 (기본 화면)
- 배지를 탭해서 태그 선택 또는 텍스트로 검색
- 조건 조정 후 결과 보기

### 4️⃣ 백업
- **설정** → **내보내기** (JSON 다운로드)
- 다른 기기에서 **가져오기** (복구)
